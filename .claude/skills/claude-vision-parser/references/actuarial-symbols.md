# 보험수리 기호 LaTeX 사전

## 생명표 기본 기호

| 기호 | LaTeX | 의미 |
|------|-------|------|
| 사망률 | `q_x` | x세의 1년간 사망확률 |
| 생존율 | `p_x` | x세의 1년간 생존확률 |
| t년 생존율 | `{}_{t}p_{x}` | x세가 t년간 생존할 확률 |
| t년 사망률 | `{}_{t}q_{x}` | x세가 t년내 사망할 확률 |
| 다중탈퇴 생존율 | `{}_{t}p_{x}^{(\tau)}` | 다중탈퇴표에서의 생존율 |
| 다중탈퇴 사망률(k원인) | `{}_{t}q_{x}^{(k)}` | k번째 원인으로의 탈퇴확률 |
| 기대여명 | `\mathring{e}_x` | x세의 완전기대여명 |
| 커튼기대여명 | `e_x` | x세의 커튼(정수)기대여명 |
| 생명표 생존수 | `l_x` | x세의 생존자 수 |
| 생명표 사망수 | `d_x` | x세의 사망자 수 |
| 연속형 사망력 | `\mu_x` | x세의 순간 사망력 |
| 생존함수 | `{}_{t}p_x = e^{-\int_0^t \mu_{x+s} ds}` | 적분형 생존확률 |

## 보험 기호

| 기호 | LaTeX | 의미 |
|------|-------|------|
| 생사혼합보험 | `A_x` | x세 가입 종신보험 APV |
| 정기보험 | `A^1_{x:\overline{n}|}` | n년 정기보험 APV |
| m회 지급 보험 | `A_x^{(m)}` | 연 m회 지급 종신보험 APV |
| 기시급 종신연금 | `\ddot{a}_x` | x세 기시급 종신연금 APV |
| 기말급 종신연금 | `a_x` | x세 기말급 종신연금 APV |
| n년 확정연금 | `\ddot{a}_{\overline{n}|}` | n년 기시급 확정연금 |
| 유기연금 | `\ddot{a}_{x:\overline{n}|}` | x세 n년 유기연금 APV |
| m회 분할 연금 | `\ddot{a}_x^{(m)}` | 연 m회 지급 종신연금 APV |

## 이자/현가 기호

| 기호 | LaTeX | 의미 |
|------|-------|------|
| 할인율 | `v = \frac{1}{1+i}` | 1년 현가인수 |
| 이력 | `\delta` | 연속복리 이자율 |

## 교환함수 (Commutation Functions)

| 기호 | LaTeX | 의미 |
|------|-------|------|
| 교환함수 C | `C_x = v^{x+1} d_x` | 사망 현가 |
| 교환함수 M | `M_x = \sum_{t=0}^{\infty} C_{x+t}` | 누적 사망 현가 |
| 교환함수 D | `D_x = v^x l_x` | 생존 현가 |
| 교환함수 N | `N_x = \sum_{t=0}^{\infty} D_{x+t}` | 누적 생존 현가 |

## 선택지리 기호

| 기호 | LaTeX | 의미 |
|------|-------|------|
| 선택지리 생존율 | `{}_{t}p_{[x]}` | 선택지리표에서의 t년 생존율 |
| 선택지리 사망수 | `l_{[x]+t}` | 선택나이 x, t년 후 생존수 |

## 특수 표기 규칙

1. **좌하 첨자**: 반드시 `{}_{t}p_{x}` 형식 (좌하 t, 우하 x)
2. **점 있는 연금**: `\ddot{a}` (ä → \ddot{a})
3. **확정기간**: `\overline{n}|` 형식 (bar over n, pipe 포함)
4. **다중탈퇴 괄호**: `^{(k)}`, `^{(\tau)}` 등 괄호 포함 상첨자
5. **기대여명 동그라미**: `\mathring{e}_x` (완전기대여명)
